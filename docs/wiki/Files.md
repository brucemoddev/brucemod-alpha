# Files & Storage

Enhanced file management and USB features.

## Enhanced File Explorer
The file explorer (SD Card and LittleFS) has been visually upgraded:
- **Icons**: Distinctive icons for folders and files.
- **Details Bar**: Shows file size and type for the selected item at the bottom.
- **Breadcrumbs**: Full path displayed in the header.

## USB Mass Storage
Turns your Cardputer into a USB drive.
- **Speedometer**: Displays average transfer speed (kB/s) updated every 20 seconds.

## Shortcuts
- **Up/Down/Arrows**: Navigate list.
- **Select (Enter)**: 
  - On Folder: Enter folder.
  - On File: Open context menu (View, Rename, Delete, etc.).
- **ESC**: Up one directory / Back to menu.
- **Letters**: Jump to file starting with that character.

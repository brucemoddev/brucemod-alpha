# UI & Customization

BruceMod features a highly customizable interface.

## Menu Styles
You can set different styles for the Main Menu and Submenus independently:
- **List**: Standard vertical list.
- **Wii**: Icon-based grid.
- **DSi**: Scrolling horizontal icons.
- **PS4**: Wide horizontal cards.
- **Vertical**: Large vertical cards.
- **C64**: Retro 8-bit aesthetic.

## Full Labels
Long menu names that are truncated in the main area are displayed in full at the bottom of the screen (except in C64 mode, where it's integrated above the cursor).

## Theming
All modules, including the **Audio Player** and **Games Menu**, respect the user's Primary and Secondary color selections.

## Shortcuts
- **ESC**: Back.
- **Arrows**: Navigation.
- **Enter**: Confirm.

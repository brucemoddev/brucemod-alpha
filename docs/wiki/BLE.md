# Bluetooth (BLE) Module

Advanced BLE utilities for sniffing and spamming.

## Features

### Sniffers
- **Wall of Flippers**: Detects Flipper Zero and other "Forbidden" packets. Optimized with pre-parsed pattern matching.
- **Wall of Airtags**: Sniffs for Apple FindMy / Airtag devices.
- **Skimmer Detector**: Detects common HC-05/06 based credit card skimmers.

### Spam / Attacks
- **Apple Spam**: Target iOS devices with proximity popups (AirPods, Setup, etc.).
- **Windows / Android / Samsung Spam**: Generic device pairing spam.
- **Airtag Spoof**: Spoof an Airtag's broadcast (Requires a previously captured MAC).

## Shortcuts
- **ESC**: Stop scan / Go back.
- **Arrows**: Scroll list of detected devices.
- **Letters (in sniffer)**: Quick jump to device by name.
